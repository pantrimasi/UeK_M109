# M109 Repo

*Ein öffentliches Repository für den **ÜK M109**, das projektbezogene Inhalte, Übungen und Ressourcen zentral bereitstellt.*

## Überblick

Dieses Repository dient als öffentliche Sammlung für Inhalte des **ÜK M109**. Ziel ist es, relevante Dateien, Aufgaben, Lösungen oder Projektinhalte übersichtlich bereitzustellen und den Zugriff für andere Personen zu vereinfachen. Das Projekt befindet sich aktuell **in Entwicklung** und wird laufend erweitert.

## Installation

1. Repository klonen:

```bash
git clone https://github.com/pantrimasi/UeK_M109
```

2. In das Projektverzeichnis wechseln:

```bash
cd UeK_M109
```

3. Projektdateien öffnen
